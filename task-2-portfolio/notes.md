## Reflection

Setting up these three platforms gave me a clear picture of how each one serves a different purpose in a student's digital journey. GitHub will be central to all my coding work — every project I build over the next four years will live here. LinkedIn is more about the professional side: making connections, following companies, and eventually applying for internships. Kaggle is where I can actually practice what I learn in class by working on real datasets and competitions. What struck me most is that having these profiles set up early — even if they are minimal right now — creates a foundation that grows with me. Employers and internship coordinators increasingly check these platforms before even looking at a resume.
