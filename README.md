## 📝 Module Summaries

### Task 1 — Digital Literacy Awareness Infographic
Created a visually engaging infographic covering what digital literacy means, useful digital tools for students, and safe internet practices. The infographic was designed to serve as an awareness resource for fellow batchmates.

📁 [View Task 1](./task-1-presentation/)

### Task 2 — Student Digital Portfolio
Set up professional profiles on **GitHub**, **LinkedIn**, and **Kaggle** to build a foundational digital portfolio. Each platform serves a distinct purpose — version control & open-source contribution, professional networking, and data science practice respectively.

📁 [View Task 2](./task-2-portfolio/)

### Task 3 — Coding & Collaboration Platforms
Explored coding practice platforms by completing a beginner-level challenge on HackerRank. Additionally, created a **Digital Literacy Awareness Quiz** using Google Forms as a collaboration exercise.

📁 [View Task 3](./task-3-platforms/)

🔗 **Google Form Quiz:** [Digital Literacy Awareness Quiz](#) *https://forms.gle/qARNABdGyEmuRN8A9*

### Task 4 — Professional Email & Etiquette Guide
Drafted two professional emails demonstrating proper email etiquette, and created a Social Media Do's and Don'ts checklist for responsible online behaviour as a college student.

📁 [View Task 4](./task-4-email-etiquette/)

### Task 5 — Cybercrime Awareness Case Study
Researched **UPI/Online Payment Fraud** — a rapidly growing cybercrime in India — and prepared a detailed case study along with an actionable "Stay Safe Online" prevention checklist.

📁 [View Task 5](./task-5-cybercrime/)
